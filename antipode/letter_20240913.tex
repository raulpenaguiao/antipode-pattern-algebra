\title{Report of changes to Antipode Formulas for Pattern Hopf algebras} % Subtitle

\author{Raul Penaguiao, Yannic Vargas}

\maketitle

We appreciate the effort and comments demonstrated by the referee to carefully review our paper.
We have taken this information to heart and overhauled the paper.

Main changes are that we rewrote the introduction chapter, as well as we included a new chapter that displays an application of this antipode formula. 

On rewritting the introduction, we were carefull in referencing that pattern Hopf algebras arise from any species with restrictions, pushed Figure 1 to chapter 3, and added pointers to specific elements of the article as suggested.

When we submitted this paper we were under the impression that we would be leaving this result for another publication, but that did not come quite to fruition.
With this in mind, we have added this application of the antipode formula as a short chapter of this paper.
Nevertheless, this should clarify the referee questions about one motivation for the result in this paper.

We also rewrote the parking function part on the abstract to reflect the content.

The typographical errors that were indicated, as well as others we have found, were dealt with:

\begin{itemize}

\item Fock function and $mathcal{K}$ $\Rightarrow$ Fock functor and $\mathcal{K}$

\item multi-quasisymmetricfunction $\Rightarrow$ multi quasisymmetric functions

\item topical $\Rightarrow$ classical

\item precise $\Rightarrow$ make precise

\item Scmitt $\Rightarrow$ Schmitt

\end{itemize}
